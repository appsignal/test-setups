Python OpenTelemetry app

  • /slow: Trigger a slow request
  • /slow_queue: Trigger a slow background task
  • /slow_queue_inline: Trigger a slow background task inline
  • /error: Trigger an error
  • /error_queue: Trigger an error background task
  • /error_queue_inline: Trigger an error background task inline
  • /make_request: Make a request to www.appsignal.com
  • /custom_instrumentation: Report custom instrumentation
  • /blog: Query and list blog posts
  • /blog/create: Create and show a new blog post
