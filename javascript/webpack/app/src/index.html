Welcome to the webpack test app

This is an example production build JavaScript app using webpack.

After making changes to the src/ directory, stop and rerun the rake javascript/webpack app:console command.

Every restart of the rake javascript/webpack app:console command will increment the AppSignal revision automatically and submit new private sourcemaps with this revision.

There is no auto reload/build available in this test app as it is specifically made to test the sourcemaps.

If everything works it should print a message below the horizontal line below.
