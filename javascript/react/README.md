# Create React App test app

Basic create-create-app test app.

## Setup

- Create a front-end app on [AppSignal.com](https://appsignal.com/redirect-to/organization?to=sites/new).
- Create an `.env` file in this directory.
  - You can use the `.example.env` file as a template.
- Paste in your API key in the `REACT_APP_APPSIGNAL_FRONTEND_KEY` environment variable.
- Start the app.
